SmartStudy Companion — User Manual

Version 1.0 • Works offline after first load • No signup.

Quick Start
  1. Enter Student name and click Save.
  2. Click Choose files and add your study material: PDF, TXT, HTML, MD, images (optional) or a big ZIP (up to 500 MB).
  3. App auto-organises files by Subject → Chapter (best results when file names contain subject or chapter numbers; still works without).
  4. Tap Create Summaries or Start Quiz.
Libraries (Class 10, Class 11…)

By default everything goes into the Default library. From the header badge (Library), you can switch to a new library (e.g., “CBSE Class 10”, “Class 11”), keep both side-by-side, archive or clear a library (coming UI in future updates). Currently this manual edition uses one active library stored in your browser.

Privacy

All processing is local to your device/browser. No files are uploaded to any server.

Tips
  • ZIP large folders to speed up adding many files.
  • For better auto-detect, include words like Math, Science, English and chapter numbers in file names.
  • Use the app on the same browser to keep your library. Clearing site data will reset it.
