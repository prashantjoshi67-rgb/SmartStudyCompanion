Smart Study Companion — User Manual

Welcome! This manual explains how to use the Smart Study Companion web app to upload study material, generate summaries and quizzes, and track progress.

1. Getting started

  1. Open the app URL in your browser (mobile or desktop).
  2. Type your name in the Name field and click Save.
  3. If you have a license for full access enter it in the License box and click Apply. Otherwise you’ll be in Demo mode.

2. Uploading material

Supported file types:

  • PDF (best)
  • Text files (.txt)
  • Images (JPG/PNG) — OCR will attempt to convert images to text
  • ZIP (optional) — if JSZip is included the app will attempt to unpack

Steps:

  1. Click the Choose Files control and pick one or multiple files.
  2. Click Process. The app will try to extract text automatically.
  3. If the PDF is a scanned image, select an OCR language in Settings and re-process.
Tip: OCR (text recognition) is CPU-heavy on mobile phones. For large book PDFs use a desktop if possible.

3. Library and file management

  • The Library tab lists all uploaded documents.
  • Click View to see the extracted text, Summary to generate a quick summary, or Delete to remove a file.
  • Use Delete all to clear your library entirely.
  • Backups: Use Manage → Export backup to download a JSON backup of your library and settings. Restore using Import.

4. Summaries and reading

Open the Summaries tab:

  • Quick — short highlight summary extracted from material.
  • Detailed — longer excerpt (multiple paragraphs).
  • Read — app will read the current summary aloud when TTS/voice is enabled.
  • Create MCQs — generates simple multiple-choice questions from the text (demo-level logic).

5. Quiz

Open the Quiz tab:

  • Choose quiz type and optional timer.
  • Quiz questions are generated from your uploaded content.
  • Each correct answer increments your score (demo scoring).

6. Voice (Text to Speech)

  1. Select a voice from the Voice dropdown. The available voices depend on your browser and OS.
  2. Click Enable to toggle voice output.
  3. When voice is enabled, summaries and quiz questions will be spoken aloud by your browser.

7. Demo vs Full Access

Demo mode is the default and shows available features. Some features or file-size limits may be restricted.

Full Access is unlocked by entering FULL-ACCESS into the license field (demo key). When active, full functionality is enabled.

8. Best practices & troubleshooting

  • No text extracted? — The PDF may be image-only (scan). In Settings choose OCR language (for example, hin for Hindi) and re-process.
  • Voice sounds foreign — try another voice in the dropdown; Chrome/Edge have the largest voice sets.
  • App seems slow — OCR and large PDFs are heavy operations on mobile. Use a desktop for large batches.
  • Files lost after clearing storage — export a backup JSON using Manage → Export before clearing browser data.

9. Where to find this manual

This manual is part of the repository and also available inside the app Help section as a link. If you need an offline copy, download the HTML or convert to PDF from your browser.
